## How to Run Locally

1.  Local Setup:
    ```bash
    git clone https://github.com/1-mukesh-1/kambaz-react-web-app.git
    cd kambaz-react-web-app
    git checkout a3
    npm install
    npm run dev
    ```
2.  Open [http://localhost:5173](http://localhost:5173)
